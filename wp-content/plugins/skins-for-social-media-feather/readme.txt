Skins for Social Media Feather adds six new skins to the Social Media Feather settings.

== Description ==

Social Media Feather is a popular social media plugin, but its styling options are limited to the available skins. This plugin provides six new skins created with the help of the Socicon generator.

== Installation ==

Skins for Social Media Feather can be installed like any other WordPress plugin. 

Once the plugin is installed and activated, the new skins will be listed with the others on the Social Media settings page and in the skin dropdowns in the Sharing and Follow Us widgets. Just select the skin you want to use and save the setting.

== Frequently Asked Questions ==

= What skins are available? =

* Color icons on a white square background
* White icons on a black square background
* White icons on a color square background
* Color icons on a white circle background
* White icons on a black circle background
* White icons on a color circle background

= Do you have any icon sets with rounded corners? =

No, but you can create your own by using one of the square sets and adding this line to your custom CSS:

.synved-social-image {border-radius: 6px;}

= What versions of Social Media Feather does this plugin work with? =

The filter hook this plugin uses was introduced in version 1.3.0 of Social Media Feather.

= What is Socicon? =

Socicon is a free social icon font with icons for dozens of social media networks. It is released under SIL Open Font License 1.1. Visit Socicon at [http://www.socicon.com](http://www.socicon.com) for details.
